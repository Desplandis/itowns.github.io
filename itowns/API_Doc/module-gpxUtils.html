(inner) load(urlFile, crs, optionsopt) → {THREE.Mesh}

Load gpx file and convert to THREE.Mesh
Parameters:
Name Type Attributes Description
urlFile string The url of gpx file
crs string The default CRS of Three.js coordinates. Should be a cartesian CRS.
options Object <optional>
 Optional properties.

Example
// How add gpx object
itowns.GpxUtils.load(url, viewer.referenceCrs).then((gpx) => {
     if (gpx) {
        viewer.scene.add(gpx);
        viewer.notifyChange(true);
     }
});
