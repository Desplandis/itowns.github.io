Fetcher

Utilitary to fetch resources from a server using the fetch API.

Static Methods

json ( url: string, options: Object ) : Promise.<Object>

Little wrapper over fetch to get some JSON.

Returns

Promise containing the JSON object.

Parameters

  • url — The URL of the resources to fetch.
  • options — Fetch options (passed directly to fetch()), see the syntax for more information.

text ( url: string, options: Object ) : Promise.<string>

Wrapper over fetch to get some text.

Returns

Promise containing the text.

Parameters

  • url — The URL of the resources to fetch.
  • options — Fetch options (passed directly to fetch()), see the syntax for more information.

texture ( url: string, options: Object ) : Promise.<THREE.Texture>

Wrapper around THREE.TextureLoader.

Returns

Promise containing the THREE.Texture.

Parameters

  • url — The URL of the resources to fetch.
  • options — Fetch options (passed directly to fetch()), see the syntax for more information. Note that THREE.js docs mentions withCredentials, but it is not actually used in THREE.TextureLoader.
