OrientedImageSource extends Source

OrientedImageSource is a specific source used to load oriented images.

Constructor

OrientedImageSource ( source: Object )

Parameters

  • source —

    Configuration object
    • source.url —

      Url for all the textures. This Url must contains {sensorId} and {cameraId}, and these pattern will be replaced to build the Url, to find the good texture for each camera for each panoramic.

Methods

extentInsideLimit ( extent: Extent ) : boolean

Tests if an extent is inside the source limits.

Returns

True if the extent is inside the limit, false otherwise.

Parameters

  • extent —

    Extent to test.

extentsInsideLimit ( extents: Array.<Extent> ) : boolean

Tests if an array of extents is inside the source limits.

Returns

True if all extents are inside, false otherwise.

Parameters

  • extents —

    Array of extents to test.

imageUrl ( cameraId: string, panoId: string ) : string

Build the url of the image, for a given panoramic id, and a given camera id.

Returns

Url of the image

Parameters

  • cameraId —

    Id of the camera
  • panoId —

    Id of the panoramic

urlFromExtent ( imageInfo: Object ) : string

Build the url of the texture, but not from extent.

Returns

Url of the image

Parameters

  • imageInfo —

    Information about the texture
    • imageInfo.camera —

      Id of the camera
    • imageInfo.pano —

      Id of the panoramic
