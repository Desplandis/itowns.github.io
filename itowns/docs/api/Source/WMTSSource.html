WMTSSource extends Source

An object defining the source of resources to get from a WMTS server. It inherits from Source.

Constructor

WMTSSource ( source: Object )

  • source — An object that can contain all properties of a WMTSSource and Source. Only url and name are mandatory.

Example

// Create the source
const wmtsSource = new itowns.WMTSSource({
    name: 'DARK',
    tileMatrixSet: 'PM',
    url: 'http://server.geo/wmts',
    format: 'image/jpg',
});

// Create the layer
const colorLayer = new itowns.ColorLayer('darkmap', {
    source: wmtsSource,
});

// Add the layer
view.addLayer(colorLayer);

Properties

isWMTSSource : boolean

Used to checkout whether this source is a WMTSSource. Default is true. You should not change this, as it is used internally for optimisation.

name : string

The name of the layer, used in the generation of the url.

version : string

The version of the WMTS server to request on. Default value is '1.0.0'.

style : string

The style to query on the WMTS server. Default value is 'normal'.

projection : string

The projection in which to fetch the data. If not specified, it is deduced from tileMatrixSet. Default value is 'EPSG:3857'.

tileMatrixSet : string

Tile matrix set of the layer, used in the generation of the url. Default value is 'WGS84'.

tileMatrixSetLimits : Object

Limits of the tile matrix set. Each limit has for key its level number, and their properties are the minTileRow, maxTileRow, minTileCol and maxTileCol.

tileMatrixSetLimits.minTileRow : number

Minimum row for tiles at the specified level.

tileMatrixSetLimits.maxTileRow : number

Maximum row for tiles at the specified level.

tileMatrixSetLimits.minTileCol : number

Minimum column for tiles at the specified level.

tileMatrixSetLimits.maxTileCol : number

Maximum column for tiles at the specified level.

zoom : Object

Object containing the minimum and maximum values of the level, to zoom in the source.

zoom.min : number

The minimum level of the source. Default value is 2.

zoom.max : number

The maximum level of the source. Default value is 20.

Methods

extentInsideLimit ( extent: Extent ) : boolean

  • extent — Extent to test.
Tests if an extent is inside the source limits.

extentsInsideLimit ( extents: Array.<Extent> ) : boolean

  • extents — Array of extents to test.
Tests if an array of extents is inside the source limits.

urlFromExtent ( extent: Extent ) : string

  • extent — Extent to convert in url.
Generates an url from an extent. This url is a link to fetch the resources inside the extent.
