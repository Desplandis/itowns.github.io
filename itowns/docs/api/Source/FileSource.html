FileSource extends Source

An object defining the source of a single resource to get from a direct access. It inherits from Source. A source of this type can only load three types of files: Geojson, GPX and KML.

Constructor

FileSource ( source: Object, crsOut: string )

  • source — An object that can contain all properties of a FileSource and Source. Only url and projection are mandatory.
  • crsOut — The projection of the output data after parsing.

Example

// Create a layer
const ariege = new itowns.GeometryLayer('ariege', new itowns.THREE.Group());

// Specify update method and conversion
ariege.update = itowns.FeatureProcessing.update;
ariege.convert = itowns.Feature2Mesh.convert({
    color: () => new itowns.THREE.Color(0xffcc00),
    extrude: () => 5000,
});

// Create and append the source
ariege.source = new itowns.FileSource({
    url: 'https://raw.githubusercontent.com/gregoiredavid/france-geojson/master/departements/09-ariege/departement-09-ariege.geojson',
    projection: 'EPSG:4326',
    format: 'application/json',
    zoom: { min: 7, max: 7 },
});

view.addLayer(ariege);
// Create the source
const gpxSource = new itowns.FileSource({
    url: 'https://raw.githubusercontent.com/iTowns/iTowns2-sample-data/master/ULTRA2009.gpx',
    projection: 'EPSG:4326',
});

// Create the layer
const gpxLayer = new itowns.ColorLayer('Gpx', {
    name: 'Ultra 2009',
    transparent: true,
    source: gpxSource,
});

// Add the layer
view.addLayer(gpxLayer);

Properties

isFileSource : boolean

Used to checkout whether this source is a FileSource. Default is true. You should not change this, as it is used internally for optimisation.

parsedData : Array

Once the file has been loaded and parsed using the GeoJsonParser, the resulting data is stored in this property.

Methods

extentInsideLimit ( extent: Extent ) : boolean

  • extent — Extent to test.
Tests if an extent is inside the source limits.

extentsInsideLimit ( extents: Array.<Extent> ) : boolean

  • extents — Array of extents to test.
Tests if an array of extents is inside the source limits.

urlFromExtent ( extent: Extent ) : string

  • extent — Extent to convert in url.
Generates an url from an extent. This url is a link to fetch the resources inside the extent.
