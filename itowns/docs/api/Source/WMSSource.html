WMSSource extends Source

An object defining the source of images to get from a WMS server. It inherits from Source.

Constructor

WMSSource ( source: Object )

  • source — An object that can contain all properties of WMSSource and Source. url, name, extent and projection are mandatory.

Example

// Create the source
const wmsSource = new itowns.WMSSource({
    url: 'https://server.geo/wms',
    version: '1.3.0',
    name: 'REGION.2016',
    style: '',
    projection: 'EPSG:3857',
    extent: {
        west: '-6880639.13557728',
        east: '6215707.87974825',
        south: '-2438399.00148845',
        north: '7637050.03850605',
    },
    transparent: true,
});

// Create the layer
const colorlayer = new itowns.ColorLayer('Region', {
    source: wmsSource,
});

// Add the layer
view.addLayer(colorlayer);

Properties

isWMSSource : boolean

Used to checkout whether this source is a WMSSource. Default is true. You should not change this, as it is used internally for optimisation.

name : string

The name of the layer, used in the generation of the url.

version : string

The version of the WMS server to request on. Default value is '1.3.0'.

style : string

The style to query on the WMS server. Default value is 'normal'.

width : number

The width of the image to fetch, in pixel. Default value is the height if set or 256.

height : number

The height of the image to fetch, in pixel. Default value is the width if set or 256.

axisOrder : string

The order of the axis, that helps building the BBOX to put in the url requesting a resource. Default value is 'wsen', other value can be 'swne'.

transparent : boolean

Tells if the image to fetch needs transparency support. Default value is false.

zoom : Object

Object containing the minimum and maximum values of the level, to zoom in the source.

zoom.min : number

The minimum level of the source. Default value is 0.

zoom.max : number

The maximum level of the source. Default value is 21.

vendorSpecific : Object

An object containing vendor specific parameters. See for example a list of these parameters for GeoServer. This object is read simply with the key being the name of the parameter and value being the value of the parameter. If used, this property should be set in the constructor parameters.

Methods

extentInsideLimit ( extent: Extent ) : boolean

  • extent — Extent to test.
Tests if an extent is inside the source limits.

extentsInsideLimit ( extents: Array.<Extent> ) : boolean

  • extents — Array of extents to test.
Tests if an array of extents is inside the source limits.

urlFromExtent ( extent: Extent ) : string

  • extent — Extent to convert in url.
Generates an url from an extent. This url is a link to fetch the resources inside the extent.
