Feature

This class improves and simplifies the construction and conversion of geographic data structures. It's an intermediary structure between geomatic formats and THREE objects.

Constructor

Feature ( type: string, crs: string, options: Object, options.buildExtent: boolean, options.withAltitude: boolean, options.withNormal: boolean )

Parameters

  • type — type of Feature. It can be 'point', 'line' or 'polygon'.
  • crs — Geographic or Geocentric coordinates system.
  • options (default: {}) — options to build feature.
    • options.buildExtent (optional) — Build extent and update when adding new vertice.
    • options.withAltitude (optional) — Set vertice altitude when adding new vertice.
    • options.withNormal (optional) — Set vertice normal when adding new vertice.

Properties

type : string

Geometry type, can be point, line, or polygon or

vertices : Array.<number>

All the vertices of the Feature.

normals : Array.<number>

All the normals of the Feature.

size : number

the number of values of the array that should be associated with a coordinates. The size is 3 with altitude and 2 without altitude.

crs : string

Geographic or Geocentric coordinates system.

geometry : Array.<FeatureGeometry>

The feature's geometry.

extent : Extent

The extent containing all the geometries composing the feature.
