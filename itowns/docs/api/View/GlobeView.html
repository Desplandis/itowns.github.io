GlobeView

Creates a view of a globe.

Constructor

GlobeView ( viewerDiv: HTMLDivElement, coordCarto: object|Coordinates, options: object )

Parameters

  • viewerDiv —

    Where to attach the view and display it in the DOM.
  • coordCarto —

    An object containing three properties: longitude, latitude and altitude. It will help placing the camera on the globe at the creation.
  • options (optional) —

    See options of View.

Example

var viewerDiv = document.getElementById('viewerDiv');
var position = new itowns.Coordinates('WGS84', 2.35, 48.8, 25e6);
var view = new itowns.GlobeView(viewerDiv, position);
var viewerDiv = document.getElementById('viewerDiv');
var position = { longitude: 2.35, latitude: 48.8, altitude: 25e6 };
var view = new itowns.GlobeView(viewerDiv, position);

Events

initialized

Fires when the view is completely loaded. Controls and view's functions can be called then.

Properties

  • target: view —

    dispatched on view
  • type: string —

    initialized

layer-added

Properties

  • target: view —

    dispatched on view
  • type: string —

    layers-added
