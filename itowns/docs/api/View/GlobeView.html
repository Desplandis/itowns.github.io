GlobeView

Creates a view of a globe.

Constructor

GlobeView ( viewerDiv: HTMLDivElement, coordCarto: object|Coordinates, options: object )

Parameters

  • viewerDiv — Where to attach the view and display it in the DOM.
  • coordCarto — An object containing three properties: longitude, latitude and altitude. It will help placing the camera on the globe at the creation.
  • options (optional) — See options of View.

Example

var viewerDiv = document.getElementById('viewerDiv');
var position = new itowns.Coordinates('WGS84', 2.35, 48.8, 25e6);
var view = new itowns.GlobeView(viewerDiv, position);
var viewerDiv = document.getElementById('viewerDiv');
var position = { longitude: 2.35, latitude: 48.8, altitude: 25e6 };
var view = new itowns.GlobeView(viewerDiv, position);

Methods

removeLayer ( layerId: string ) : boolean

Removes a specific imagery layer from the current layer list. This removes layers inserted with attach().

Parameters

  • layerId — The identifier

Example

view.removeLayer('layerId');

Events

initialized

Fires when the view is completely loaded. Controls and view's functions can be called then.

Properties

  • target: view — dispatched on view
  • type: string — initialized

layer-added

Fires when a layer is added

Properties

  • layerId: string — the id of the layer
  • target: view — dispatched on view
  • type: string — layers-added

layer-removed

Fires when a layer is removed

Properties

  • layerId: string — the id of the layer
  • target: view — dispatched on view
  • type: string — layers-added

layers-order-changed

Fires when the layers oder has changed

Properties

  • new: object
    • new.sequence: array
      • new.sequence.0: number — the new layer at position 0
      • new.sequence.1: number — the new layer at position 1
      • new.sequence.2: number — the new layer at position 2
  • previous: object
    • previous.sequence: array
      • previous.sequence.0: number — the previous layer at position 0
      • previous.sequence.1: number — the previous layer at position 1
      • previous.sequence.2: number — the previous layer at position 2
  • target: view — dispatched on view
  • type: string — layers-order-changed
