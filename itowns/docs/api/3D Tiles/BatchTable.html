BatchTable

Class representing a 3D Tiles batch table

Constructor

BatchTable ( json: Object, binary: ArrayBuffer, batchLength: number )

Parameters

  • json —

    batch table json part
  • binary —

    buffer representing the batch table
  • batchLength —

    the length of the batch. binary part (not supported yet)

Methods

getPickingInfo ( featureId: integer ) : Object

Creates and returns a javascript object holding the displayable information from the batch table and from extensions of the batch table for a given feature

Returns

  • displayable information relative to the batch table and its extensions for the feature with id=featureId. Object is formatted as follow: {BatchTable: {BatchTableFeatureProperties}, ExtensionName: {ExtensionDisplayableInfo}}

Parameters

  • featureId —

    id of the feature

removeExtensionFromContent ( extensionName: string )

Remove an extension from this.content. Must be called when an extension has been parsed and added to this.extensions

Parameters

  • extensionName —

    the name of the extension to remove
