$3dtilesLayer

Class for $3dtilesLayer.

Constructor

$3dtilesLayer ( )

Example

var layer = new itowns.GeometryLayer('3dtiles-example');
layer.protocol = '3d-tiles'
layer.url = 'http://example/tileset.json';

Properties

url : string

tileset.json URL

overrideMaterial : boolean|THREE.Material

override meshes material embedded in the binary files and use a default one instead.

material : THREE.Material

material cloned and assigned each time a points mesh is created.

sseThreshold : number

s(creen) s(pace) e(rror) threshold in pixels. Define how many pixels the geometricError from a tile must cover to kick the subdivision mechanism in.

cleanupDelay : number

delay in milliseconds after which an undisplayed tiles will be removed from memory.

onTileContentLoaded : function

callback called when new content is added.
