$3dTilesAbstractExtension

Abstract class for 3DTiles Extensions. Extensions implemented by the user must inherit from this class. Example of extension extending this class: BatchTableHierarchyExtension

Methods

getPickingInfo ( featureId: integer )

  • featureId — id of the feature
Method to get the displayable information related to a given feature from an extension. All extensions must implement it (event if it returns an empty object).
