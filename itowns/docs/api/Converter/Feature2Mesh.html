Feature2Mesh

Static Methods

convert ( options: Object, options.altitude: number|function, options.extrude: number|function, options.color: object|function, options.batchId: function ) : function

  • options — options controlling the conversion
    • options.altitude — define the base altitude of the mesh
    • options.extrude — if defined, polygons will be extruded by the specified amount
    • options.color — define per feature color
    • options.batchId (optional) — optional function to create batchId attribute. It is passed the feature property and the feature index. As the batchId is using an unsigned int structure on 32 bits, the batchId could be between 0 and 4,294,967,295.
Return a function that converts Features to Meshes. Feature collection will be converted to a a THREE.Group.

Example

Example usage of batchId with featureId.
view.addLayer({
    id: 'WFS Buildings',
    type: 'geometry',
    update: itowns.FeatureProcessing.update,
    convert: itowns.Feature2Mesh.convert({
        color: colorBuildings,
        batchId: (property, featureId) => featureId,
        altitude: altitudeBuildings,
        extrude: extrudeBuildings }),
    onMeshCreated: function scaleZ(mesh) {
        mesh.scale.z = 0.01;
        meshes.push(mesh);
    },
    filter: acceptFeature,
    source,
});
Example usage of batchId with property.
view.addLayer({
    id: 'WFS Buildings',
    type: 'geometry',
    update: itowns.FeatureProcessing.update,
    convert: itowns.Feature2Mesh.convert({

        batchId: (property, featureId) => property.house ? 10 : featureId,
