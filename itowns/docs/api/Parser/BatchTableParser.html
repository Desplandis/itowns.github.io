BatchTableParser

Static Methods

parse ( buffer: ArrayBuffer, BTBinaryLength: integer, BATCH_LENGTH: number ) : Promise

Parse a batch table buffer and returns a promise that resolves with a BatchTable object.

Returns

  • a promise that resolves with a BatchTable object.

Parameters

  • buffer —

    the batch table buffer.
  • BTBinaryLength —

    length of the binary part of the batch table
  • BATCH_LENGTH —

    the length of the batch.
