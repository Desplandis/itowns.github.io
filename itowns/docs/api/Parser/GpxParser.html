GpxParser

The GpxParser module provides a parse method that takes a GPX in and gives an object formatted for iTowns containing all necessary informations to display this GPX.

Static Methods

parse ( gpx: XMLDocument, options: GeoJsonParser~GeoJsonParserOptions ) : Promise

Parse a GPX file content and return a FeatureCollection.

Returns

A promise resolving with a FeatureCollection.

Parameters

  • gpx — The GPX file content to parse.
  • options — Options controlling the parsing.
