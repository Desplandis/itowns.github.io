GpxParser

Static Methods

parse ( xml: string, options: Object, options.crs: string, options.enablePin: boolean, options.networkOptions: NetworkOptions, options.lineWidth: number ) : THREE.Mesh

  • xml — the gpx file or xml.
  • options (optional) — additional properties.
    • options.crs — the default CRS of Three.js coordinates. Should be a cartesian CRS.
    • options.enablePin (default: true) — draw pin for way points.
    • options.networkOptions (optional) — options for fetching resources over network.
    • options.lineWidth (default: 12) — set line width to track line.
Parse gpx file and convert to THREE.Mesh

Example

// How to add a gpx object
itowns.GpxParser.parse(file, { crs: viewer.referenceCrs }).then((gpx) => {
     if (gpx) {
        viewer.scene.add(gpx);
        viewer.notifyChange();
     }
});
