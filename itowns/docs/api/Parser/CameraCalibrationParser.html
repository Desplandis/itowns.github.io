CameraCalibrationParser

The CameraCalibrationParser module provide a parse method that takes a JSON array of camera calibrations in and yields an array of OrientedImageCamera

Static Methods

parse ( json: string|JSON, options: Object, options.near: string, options.far: string ) : Promise

Parser a JSON array of camera calibrations and return an array of OrientedImageCamera.

Returns

A promise resolving with an array of OrientedImageCamera.

Parameters

  • json — the json content of the calibration file.
  • options (default: {}) — Options controlling the parsing.
    • options.near (default: 0.1) — Near of the created cameras. Default value comes from created OrientedImageCamera
    • options.far (default: 1000) — Far of the created cameras. Default value comes from created OrientedImageCamera

Types

CameraCalibrationJson : Object

  • projection: Array.<number> — projection matrix,
  • size: Array.<number> — image size in pixel.
  • position: Array.<number> — position of the camera.
  • rotation: Array.<number> — rotation matrix
  • distorsion: Object (default: {}) — distorsion
    • distorsion.pps: Array.<number> (optional)
    • distorsion.poly357: Array.<number> (optional)
    • distorsion.limit: Array.<number> (optional)
    • distorsion.l1l2: Array.<number> (optional)
    • distorsion.etat: Array.<number> (optional)
Description of a camera calibration in a JSON file.
