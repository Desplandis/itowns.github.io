VectorTileParser

Static Methods

parse ( file: ArrayBuffer, options: Object ) : Promise

Parse a vector tile file and return a Feature or an array of Features. While multiple formats of vector tile are available, the only one supported for the moment is the Mapbox Vector Tile.

Returns

A Promise resolving with a Feature or an array a Features.

Parameters

  • file — The vector tile file to parse.
  • options — Options controlling the parsing.
    • options.extent — The Extent to convert the input coordinates to.
    • options.coords — Coordinates of the layer.
    • options.filteringExtent (optional) — Optional filter to reject features outside of this extent.
    • options.mergeFeatures (default: true) — If true all geometries are merged by type and multi-type
    • options.withNormal (default: true) — If true each coordinate normal is computed
    • options.withAltitude (default: true) — If true each coordinate altitude is kept
    • options.filter (optional) — Filter function to remove features.
    • options.isInverted (optional) — This option is to be set to the correct value, true or false (default being false), if the computation of the coordinates needs to be inverted to same scheme as OSM, Google Maps or other system. See this link for more informations.
