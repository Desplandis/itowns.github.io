parse ( buffer: ArrayBuffer, options: Object, options.gltfUpAxis: string, options.urlBase: string, options.doNotPatchMaterial: boolean, options.opacity: float, options.overrideMaterials: boolean|Material ) : Promise

Parse b3dm buffer and extract THREE.Scene and batch table

Returns

a promise that resolves with an object containig a THREE.Scene (gltf) and a batch table (batchTable).

Parameters

  • buffer — the b3dm buffer.
  • options — additional properties.
    • options.gltfUpAxis (default: 'Y') — embedded glTF model up axis.
    • options.urlBase — the base url of the b3dm file (used to fetch textures for the embedded glTF model).
    • options.doNotPatchMaterial (default: 'false') — disable patching material with logarithmic depth buffer support.
    • options.opacity (default: 1.0) — the b3dm opacity.
    • options.overrideMaterials (default: 'false') — override b3dm's embedded glTF materials. If overrideMaterials is a three.js material, it will be the material used to override.
