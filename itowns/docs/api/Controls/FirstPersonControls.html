FirstPersonControls

Constructor

FirstPersonControls ( view: View, options: object, options.focusOnClick: boolean, options.focusOnMouseOver: boolean, options.moveSpeed: boolean, options.verticalFOV: number, options.panoramaRatio: number, options.disableEventListeners: boolean )

Parameters

  • view
  • options
    • options.focusOnClick — whether or not to focus the renderer domElement on click
    • options.focusOnMouseOver — whether or not to focus when the mouse is over the domElement
    • options.moveSpeed — if > 0, pressing the arrow keys will move the camera
    • options.verticalFOV — define the max visible vertical angle of the scene in degrees (default 180)
    • options.panoramaRatio — alternative way to specify the max vertical angle when using a panorama. You can specify the panorama width/height ratio and the verticalFOV will be computed automatically
    • options.disableEventListeners — if true, the controls will not self listen to mouse/key events. You'll have to manually forward the events to the appropriate functions: onMouseDown, onMouseMove, onMouseUp, onKeyUp, onKeyDown and onMouseWheel.

Methods

reset ( preserveRotationOnX: boolean )

Resets the controls internal state to match the camera' state. This must be called when manually modifying the camera's position or rotation.

Parameters

  • preserveRotationOnX — if true, the look up/down rotation will not be copied from the camera

update ( dt: number, updateLoopRestarted: boolean, force: boolean )

Updates the camera position / rotation based on occured input events. This is done automatically when needed but can also be done if needed.

Parameters

  • dt — ellpased time since last update in seconds
  • updateLoopRestarted — true if itowns' update loop just restarted
  • force — set to true if you want to force the update, even if it appears unneeded.
