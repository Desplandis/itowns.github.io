StateControl

It represents the control's states. Each state is a control mode of the camera and how to interact with the interface to activate this mode.

Methods

inputToState ( mouseButton: Number, keyboard: Number ) : state

  • mouseButton — The mouse button
  • keyboard — The keyboard

touchToState ( finger: Number ) : state

  • finger — The number of finger
get the state corresponding to the number of finger on the pad
