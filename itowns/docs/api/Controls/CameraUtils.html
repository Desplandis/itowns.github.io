CameraUtils

Static Methods

animateCameraToLookAtTarget ( view: View, camera: Camera, params: cameraTransformOptions ) : Promise

  • view — The camera view
  • camera — The camera to animate
  • params — The parameters

getDiffParams ( first: cameraTransformOptions, second: cameraTransformOptions ) : object

  • first — param to compare with the second
  • second — param to compare with the first
Gets the difference camera transformation

getTransformCameraLookingAtTarget ( view: View, camera: Camera ) : cameraTransformOptions

  • view — The camera view
  • camera — The camera to get transform
Gets the current parameters transform camera looking at target.

sequenceAnimationsToLookAtTarget ( view: View, camera: Camera, params: Array.<cameraTransformOptions> ) : Promise

  • view — The camera view
  • camera — The camera to animate
  • params — array parameters, each parameters transforms are apply to camera, in serial
chain animation transform to camera

stop ( view: View, camera: Camera )

  • view — The camera view
  • camera — The camera to stop animation
Stop camera's animation

transformCameraToLookAtTarget ( view: View, camera: Camera, params: cameraTransformOptions ) : Promise

  • view — The camera view
  • camera — The camera to transform
  • params — The parameters
Apply transform to camera
