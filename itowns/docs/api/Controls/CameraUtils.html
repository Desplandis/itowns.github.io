transformCameraToLookAtTarget ( view: View, camera: Camera, params: CameraUtils~CameraTransformOptions ) : Promise

Apply transform to camera

Returns

promise with resolve final CameraUtils~CameraTransformOptions

Parameters

  • view — The camera view
  • camera — The camera to transform
  • params — The parameters

Types

CameraTransformOptions : Object

  • coord: Coordinate (default: currentCoordinate) — Camera look at geographic coordinate
  • tilt: Number (default: currentTilt) — camera's tilt, in degree
  • heading: Number (default: currentHeading) — camera's heading, in degree
  • range: Number (default: currentRange) — camera distance to target coordinate, in meter
  • time: Number (default: 2500) — duration of the animation, in ms
  • proxy: boolean (default: true) — use proxy to handling camera's transformation. if proxy == true, other camera's transformation stops rig's transformation
  • easing: Number (default: TWEEN.Easing.Quartic.InOut) — in and out easing animation
  • callback: function (optional) — callback call each animation's frame (params are current cameraTransform and worldTargetPosition)
  • stopPlaceOnGroundAtEnd: boolean (default: defaultStopPlaceOnGroundAtEnd) — stop place target on the ground at animation ending
