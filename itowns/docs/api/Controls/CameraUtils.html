transformCameraToLookAtTarget ( view: View, camera: Camera, params: CameraUtils~CameraTransformOptions ) : Promise

  • view — The camera view
  • camera — The camera to transform
  • params — The parameters
Apply transform to camera

Types

CameraTransformOptions : Object

  • coord (default: currentCoordinate) — Camera look at geographic coordinate
  • tilt (default: currentTilt) — camera's tilt, in degree
  • heading (default: currentHeading) — camera's heading, in degree
  • range (default: currentRange) — camera distance to target coordinate, in meter
  • time (default: 2500) — duration of the animation, in ms
  • proxy (default: true) — use proxy to handling camera's transformation. if proxy == true, other camera's transformation stops rig's transformation
  • easing (default: TWEEN.Easing.Quartic.InOut) — in and out easing animation
  • callback (optional) — callback call each animation's frame (params are current cameraTransform and worldTargetPosition)
  • stopPlaceOnGroundAtEnd (default: defaultStopPlaceOnGroundAtEnd) — stop place target on the ground at animation ending
