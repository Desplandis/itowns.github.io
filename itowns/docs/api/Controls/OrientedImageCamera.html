OrientedImageCamera

OrientedImageCamera is a ThreeJs camera adapted to photogrammetric description. So we can build a ThreeJs perspective camera from size and focal information.

Constructor

OrientedImageCamera ( size: number|Vector2, focal: number|Vector2, center: Vector2, near: number, far: number, skew: number, aspect: number )

Parameters

  • size (default: 1024) — image size in pixels (default: x=1024, y=x)
  • focal (default: 1024) — focal length in pixels (default: x=1024, y=x)
  • center — principal point in pixels (default: size/2)
  • near (default: 0.1) — Camera frustum near plane (default: see THREE.PerspectiveCamera).
  • far (default: 10000) — Camera frustum far plane (default: see THREE.PerspectiveCamera).
  • skew — shear transform parameter (default: 0)
  • aspect — aspect ratio of the camera (default: size.x/size.y).
