PlanarControls

Camera controls adapted for a planar view, with animated movements. Usage is as follow:
  • Left mouse button: drag the camera (translation on the (xy) world plane).
  • Right mouse button: pan the camera (translation on the vertical (z) axis of the world plane).
  • CTRL + Left mouse button: rotate the camera around the focus point.
  • Wheel scrolling: zoom toward the cursor position.
  • Wheel clicking: smart zoom toward the cursor position (animated).
  • Y key: go to the starting view (animated).
  • T key: go to the top view (animated).
This controls needs to be instantiated after the camera setup.

Constructor

PlanarControls ( view: PlanarView, options: PlanarControls~PlanarControlsOptions )

  • view
  • options (optional)

Example

// From the orthographic example
new itowns.PlanarControls(view, {
    // We do not want the user to zoom out too much
    maxAltitude: 40000000,
    // We want to keep the rotation disabled, to only have a view from the top
    enableRotation: false,
    // Faster zoom in/out speed
    zoomInFactor: 0.5,
    zoomOutFactor: 0.5,
    // Don't zoom too much on smart zoom
    smartZoomHeightMax: 100000,
});

Types

PlanarControlsOptions : Object

  • enableRotation: boolean (default: true) — Enable the rotation with the CTRL + Left mouse button and in animations, like the smart zoom.
  • rotateSpeed: number (default: 2.0) — Rotate speed.
  • maxPanSpeed: number (default: 15) — Pan speed when close to maxAltitude.
  • minPanSpeed: number (default: 0.05) — Pan speed when close to the ground.
  • zoomTravelTime: number (default: 0.2) — Animation time when zooming.
  • zoomInFactor: number (default: 0.25) — Zoom movement is equal to the distance to the zoom target, multiplied by this factor when zooming in.
  • zoomOutFactor: number (default: 0.4) — Zoom movement is equal to the distance to the zoom target, multiplied by this factor when zooming out.
  • maxAltitude: number (default: 12000) — Maximum altitude reachable when panning.
  • groundLevel: number (default: 200) — Minimum altitude reachable when panning.
  • autoTravelTimeMin: number (default: 1.5) — Minimum duration for animated travels with the auto parameter.
  • autoTravelTimeMax: number (default: 4) — Maximum duration for animated travels with the auto parameter.
  • autoTravelTimeDist: number (default: 20000) — Maximum travel distance for animated travel with the auto parameter.
  • smartZoomHeightMin: number (default: 75) — Minimum height above ground reachable after a smart zoom.
  • smartZoomHeightMax: number (default: 500) — Maximum height above ground reachable after a smart zoom.
  • instantTravel: boolean (optional) — If set to true, animated travels will have no duration.
  • minZenithAngle: number (optional) — The minimum reachable zenith angle for a camera rotation, in degrees.
  • maxZenithAngle: number (default: 82.5) — The maximum reachable zenith angle for a camera rotation, in degrees.
  • focusOnMouseOver: boolean (default: true) — Set the focus on the canvas if hovered.
  • focusOnMouseClick: boolean (default: true) — Set the focus on the canvas if clicked.
  • handleCollision: boolean (default: true)
Options for the instantiation of a PlanarControls.
