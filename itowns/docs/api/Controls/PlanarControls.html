PlanarControls

Planar controls is a camera controller adapted for a planar view, with animated movements. Usage is as follow :

  • Left mouse button: drag the camera (translation on the (xy) world plane).
  • Right mouse button: pan the camera (translation on the vertical (z) axis of the world plane).
  • CTRL + Left mouse button: rotate the camera around the focus point.
  • Wheel scrolling: zoom toward the cursor position.
  • Wheel clicking: smart zoom toward the cursor position (animated).
  • Y key: go to the starting view (animated).
  • T key: go to the top view (animated).

Constructor

PlanarControls ( view: PlanarView, options: object )

Parameters

  • view —

    the view where the controls will be used
  • options
    • options.enableRotation (default: true) —

      Enable the rotation with the CTRL + Left mouse button and in animations, like the smart zoom.
    • options.rotateSpeed (default: 2.0) —

      Rotate speed.
    • options.maxPanSpeed (default: 15) —

      Pan speed when close to maxAltitude.
    • options.minPanSpeed (default: 0.05) —

      Pan speed when close to the ground.
    • options.zoomTravelTime (default: 0.2) —

      Animation time when zooming.
    • options.zoomInFactor (default: 0.025) —

      Zoom movement is equal to the distance to the zoom target, multiplied by this factor when zooming in.
    • options.zoomOutFactor (default: 0.4) —

      Zoom movement is equal to the distance to the zoom target, multiplied by this factor when zooming out.
    • options.maxAltitude (default: 12000) —

      Maximum altitude reachable when panning.
    • options.groundLevel (default: 200) —

      Minimum altitude reachable when panning.
    • options.autoTravelTimeMin (default: 1.5) —

      Minimum duration for animated travels with the auto parameter.
    • options.autoTravelTimeMax (default: 4) —

      Maximum duration for animated travels with the auto parameter.
    • options.autoTravelTimeDist (default: 20000) —

      Maximum travel distance for animated travel with the auto parameter.
    • options.smartZoomHeightMin (default: 75) —

      Minimum height above ground reachable after a smart zoom.
    • options.smartZoomHeightMax (default: 500) —

      Maximum height above ground reachable after a smart zoom.
    • options.instantTravel (optional) —

      If set to true, animated travels will have no duration.
    • options.minZenithAngle (optional) —

      The minimum reachable zenith angle for a camera rotation, in degrees.
    • options.maxZenithAngle (default: 82.5) —

      The maximum reachable zenith angle for a camera rotation, in degrees.
    • options.focusOnMouseOver (default: true) —

      Set the focus on the canvas if hovered.
    • options.focusOnMouseClick (default: true) —

      Set the focus on the canvas if clicked.
    • options.handleCollision (default: true)
    • options.enableSmartTravel (default: true) —

      enable smart travel
