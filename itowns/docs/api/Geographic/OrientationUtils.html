OrientationUtils

Static Methods

globeQuaternionFromAttitude ( attitude: Attitude, coordinate: Coordinates, target: THREE.Quaternion ) : THREE.Quaternion

  • attitude — Attitude with properties: (omega, phi, kappa), (roll, pitch, heading) or none.
  • coordinate — position on the globe
  • target — Quaternion to set

localQuaternionFromAttitude ( attitude: Attitude, target: THREE.Quaternion ) : THREE.Quaternion

  • attitude — Attitude with properties: (omega, phi, kappa), (roll, pitch, heading) or none. Note that convergence of the meridians is not taken into account.
  • target — Quaternion to set

quaternionFromAttitude ( attitude: Attitude, coordinate: Coordinates, needsENUFromGeodesicNormal: Boolean, target: THREE.Quaternion ) : THREE.Quaternion

  • attitude — Attitude with properties: (omega, phi, kappa), (roll, pitch, heading) or none.
  • coordinate — position the oject (used to apply another rotation on Globe CRS)
  • needsENUFromGeodesicNormal — should be true on globe CRS. If true, we will apply another rotation : The rotation use to create ENU local space at coordinate parameter position.
  • target — Quaternion to set
Read rotation information (roll pitch heading or omega phi kappa), Create a ThreeJs quaternion representing a rotation.

Types

Attitude : Object

  • omega: Number — angle in degrees
  • phi: Number — angle in degrees
  • kappa: Number — angle in degrees
  • roll: Number — angle in degrees
  • pitch: Number — angle in degrees
  • heading: Number — angle in degrees
