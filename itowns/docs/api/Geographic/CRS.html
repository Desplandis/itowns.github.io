CRS

This module provides basic methods to manipulate a CRS (as a string).

Static Methods

isGeographic ( crs: string ) : boolean

Assert that the CRS is geographic.

Parameters

  • crs — The CRS to validate.

isMetricUnit ( crs: string ) : boolean

Assert that the CRS is using metric units.

Parameters

  • crs — The CRS to validate.

isValid ( crs: string )

Assert that the CRS is valid one.

Parameters

  • crs — The CRS to validate.

reasonnableEpsilon ( crs: string ) : number

Give a reasonnable epsilon to use with this CRS.

Returns

0.01 if the CRS is EPSG:4326, 0.001 otherwise.

Parameters

  • crs — The CRS to use.
