Class: GlobeView

GlobeView

new GlobeView(viewerDiv, coordCarto, optionsopt)

Creates the viewer Globe (the globe of iTowns). The first parameter is the coordinates on wich the globe will be centered at the initialization. The second one is the HTML div in wich the scene will be created.
Parameters:
Name Type Attributes Description
viewerDiv HTMLDivElement Where to instanciate the Three.js scene in the DOM
coordCarto object
options object <optional>

getLayers(filter) → {Array.<Layer>}

Get all layers, with an optionnal filter applied. The filter method will be called with 2 args: - 1st: current layer - 2nd: (optional) the geometry layer to which the current layer is attached
Parameters:
Name Type Description
filter function
Inherited From:
  • View#getLayers
Source:
  • Core/View.js, line 266
Returns:
Type
Array.<Layer>
Example
// get all layers
view.getLayers();
// get all color layers
view.getLayers(layer => layer.type === 'color');
// get all elevation layers
view.getLayers(layer => layer.type === 'elevation');
// get all geometry layers
view.getLayers(layer => layer.type === 'geometry');
// get one layer with id
view.getLayers(layer => layer.id === 'itt');
